2. Description of Service

MailArmour provides inbox security tools to identify phishing attempts, analyze email sender details, perform domain age checks, flag suspicious redirecting links, and compile email analysis reports inside your browser client. The analysis is performed by transferring email metadata to our secure cloud server using Anthropic Claude.

3. Free and Pro Tiers

We offer two service levels for our users:

  • Free Shield: Restricts active inbox security checks to a limit of 10 scans per lifetime/installation. Features include Hindi localization, standard sender validation, and basic attachment check.
  • Pro Shield: Unlocks unlimited inbox scans, access to domain age check, whitelisting, history logs of up to 50 scans, and exports (PDF, CSV, JSON reports).
